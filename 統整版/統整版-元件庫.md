# 第壹部分 · 共用設計系統（Vercel/Geist）

## A. 共用 Design Token（Vercel/Geist・統一基準）

兩款遊戲皆以此為**風格統一基準**。設計語言：近白畫布 `#fafafa`、墨黑 `#171717`、1px hairline `#ebebeb`、單一藍點綴 `#0070f3`。Geist 不可用時以 **Inter** 替代（維持 600 標題、500 標籤/按鈕）。

### 色彩

| Token | HEX | 用途 |
|---|---|---|
| ink / primary | `#171717` | 主文字、莊 Banker、primary 填色、選中、CTA、邊框最深階 |
| body | `#4d4d4d` | 次要內文 |
| mute | `#8f8f8f` | 說明文字、和 Tie、metadata |
| faint | `#a1a1a1` | placeholder、disabled |
| hairline | `#ebebeb` | 1px 卡片／分隔線 |
| hairline-soft | `#f2f2f2` | 淺灰面板 |
| canvas | `#fafafa` | 頁面底色 |
| canvas-elevated | `#ffffff` | 卡片、按鈕、輸入框 |
| link / accent | `#0070f3` | 閒 Player、連結、進行中狀態、focus、已鎖定提示 |

**語意色（跨遊戲共用）**：閒 Player = 藍 `#0070f3`、莊 Banker = 墨黑 `#171717`、和 Tie = 中灰 `#8f8f8f`。

### 字體

| Style | Family / Weight | Size / LH / Tracking | 用途 |
|---|---|---|---|
| heading-md | Geist SemiBold (600) | 20 / 28 / −0.4 | 卡片標題 |
| amount | Geist SemiBold (600) | 18 / − / −0.4 | 金額 |
| label-sm | Geist Medium (500) | 14 / 20 / −0.28 | 強調標籤 |
| mono-eyebrow | Geist Mono Medium (500) | 12 / 16 / 0 | 大寫技術 eyebrow（LIVE、狀態） |
| body-md | Geist Regular (400) | 14 / 20 / 0 | 內文、賠率 |
| body-sm | Geist Regular (400) | 12 / 16 / 0 | 註腳、metadata |
| button-md | Geist Medium (500) | 14 / 20 / 0 | 按鈕文字 |

### 圓角 / 間距

- 圓角：`sm 6px`（按鈕/輸入/狀態列/路紙格/邊注 tile）、`md 12px`（卡片）、`lg 16px`（大面板）、`full 9999px` / `pill 100`（圓形圖示鈕、籌碼膠囊）
- 間距（4px 基準）：`xxs 4 · xs 8 · sm 12 · md 16 · lg 24 · xl 32`

# 第參部分 · Blackjack（21 點）元件

> 本部分為線框（wireframe）階段的 UI 元件清單，整理各元件的內容、尺寸與樣式 token，供開發與設計交接使用。
> **風格統一原則**：以 Vercel/Geist（第壹部分 A）為目標樣式；[第二節元件清單](#二元件清單線框階段)保留線框階段實際色值以利對照，其對映關係見[第三節](#三設計-token-彙整線框舊值--vercel-對映)，Vercel 化後的成品版本見[第四節](#四vercel-元件區塊geist-風格)。

## 一、版面總覽（Blackjack）

整個畫板由上而下分為三大區塊：

- **頂部資訊區**：荷官／視訊區域、Betting Panel、Roadmap & Stat 三個大面板並排，下方接 Side Bets（邊注）列。
- **中央遊戲區**：左側為選單列／聊天室／路紙統計／結算資訊；中央為保險詢問、座位群、計時器、玩家牌、荷官牌；右側為籌碼區、Open Cards 按鈕。
- **底部狀態區**：結果 Toast（Win）、Cash Out 按鈕、「已開局,請投注」提示。
